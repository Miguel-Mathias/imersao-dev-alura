Base de Conhecimento Temática: The Originals
Este projeto Front-End (HTML, CSS, JavaScript) é uma base de conhecimento interativa sobre a série The Originals, demonstrando manipulação robusta de dados e design temático.

Utilidade & Eficácia: Implementação de busca eficiente (fetch, .filter() case-insensitive) sobre dados simulados em JSON, com carga assíncrona para melhor desempenho. O controle de estado garante que a tela inicie limpa, otimizando a experiência do usuário.

Criatividade & Apresentação: O design aplica um Dark Theme sofisticado com cores temáticas (vermelho sangue) e elementos visuais criativos. Destaque para o favicon e o elemento de "gota de sangue" animada na inicialização, que elevam a apresentação e criam uma forte identidade visual.
